## Component Dimensions & Layout

### Header
- **Height**: 60-70px (normal), 50-60px (sticky)
- **Sticky**: Yes, on scroll > 50px
- **Breakpoints**: Desktop (lg:), Tablet (md:), Mobile
- **Z-Index**: 50 (sticky), 40 (normal)

### Hero Section
- **Height**: 100vh (full-height option)
- **Min Height**: 500px (small screens), 600px (tablet)
- **Background**: Full-width image/gradient
- **Layout**: Centered content

### Cards Grid
- **Desktop**: 4 columns (lg:grid-cols-4)
- **Tablet**: 2 columns (md:grid-cols-2)
- **Mobile**: 1 column
- **Gap**: 6 units (1.5rem)

### Testimonial Carousel
- **Width**: max-w-3xl (48rem)
- **Background**: White with shadow
- **Padding**: 8-12 units
- **Navigation**: Previous/Next buttons + dots

### Feature Grid (FeatureHighlight)
- **Desktop**: 2 columns (1 text, 1 image)
- **Tablet**: 1 column
- **Gap**: 8 units (2rem)

### Blog Grid
- **Desktop**: 4 columns (lg:grid-cols-4)
- **Tablet**: 2 columns (md:grid-cols-2)
- **Mobile**: 1 column
- **Card Height**: 400-500px

## Interactive Elements

### Buttons
```
Primary CTA:
  Background: #2563eb → #1e40af (hover)
  Padding: py-3 px-8
  Border Radius: rounded-lg
  Shadow: shadow-md → shadow-lg (hover)
  Text: white, bold, centered

Secondary/Ghost:
  Transparent with border or text-only
  Hover: background color change
```

### Card Hover Effects
```
hover:-translate-y-2    (lift up 2px)
hover:shadow-xl         (increase shadow)
transition-all          (smooth)
duration-300            (300ms)
```

### Dropdown Menus
```
Normal:     opacity-0 invisible
On Hover:   opacity-100 visible
Transition: 200ms smooth
Z-Index:    50+ to appear above content
```

## Animation Effects

### Fade In Up
```css
From: opacity-0, translateY(20px)
To:   opacity-1, translateY(0)
Duration: 600ms
Timing: ease-out
```

### Slide In Right
```css
From: opacity-0, translateX(20px)
To:   opacity-1, translateX(0)
Duration: 600ms
Timing: ease-out
```

### Pulse Glow (for CTAs)
```css
Box-shadow animation
0-100%: box-shadow 0 0 20px → 30px
Duration: 2 seconds infinite
Timing: ease-in-out
```
